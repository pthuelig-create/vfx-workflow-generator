04 · Naming Convention

▼
Marker Naming (Editorial)
🎬 VFX-Shots werden in der Avid/Premiere-Timeline nach dem Shot-Namensschema gemarkert: PROJ_E101_001_010 (Show Code + Episode + Scene + Shot). Das Namensschema wird automatisch aus der Naming Convention übernommen.
📌 Scene = Scene number from the screenplay / script (3 digits, e.g. 001, 042, 115). Shot = shot number within that scene, in steps of 10 (010, 020 …). Version numbers must be unique and never reused. QT and EXR always share the same version number.
